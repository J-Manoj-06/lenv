## File Size Tracking

### Upload Process
```
File selected → Read bytes → Calculate size
→ uploadBytes.length = fileSize
→ Stored in MediaMetadata
→ Sent to Firestore
```

### Display Process
```
Message loaded → Get metadata.fileSize
→ Pass to MediaPreviewCard
→ Display: "2.0 MB"
```

### Download Process
```
Download completes → Verify file size
→ Get actual file.length()
→ Update metadata
→ Display correct size
```

## If Download Fails

Check console for error:

| Error | Cause | Fix |
|-------|-------|-----|
| HTTP 404 | File not in R2 | Check Cloudflare Worker |
| HTTP 500 | Server error | Restart Worker |
| Permission denied | Storage permission | Settings → Allow Storage |
| No space | Disk full | Delete other files |
| No network | Offline | Check internet |

## 📍 STEP BY STEP TO FIND IT

### Step 1: Open Your App
- Launch the New Reward app

### Step 2: Login as Student
- Use any student credentials
- You'll land on the **Student Dashboard** (home page)

### Step 3: Look at Top-Right Corner
The top-right corner shows:
```
[🔥 25] [🔧] [👤]
  ↑      ↑    ↑
  |      |    └─ Profile Button (existing)
  |      └────── Dev Tools Button (NEW!)
  └────────────── Streak Badge (existing)
```

### Step 4: Click the Wrench Icon (🔧)
- The **orange wrench icon** in the top-right
- This opens the Dev Tools screen

### Step 5: Scroll Down in Dev Tools
- You'll see several buttons:
  - Seed Schools (blue)
  - Seed Test Data (green)
  - Clear Test Data (red)
  - Open Latest Result (orange)
  - **🎥 Test Media Upload (green)** ← CLICK THIS ONE
  - Info Card (bottom)

---

## 🎯 SUCCESS
If you can see the **orange wrench icon** (🔧) in the top-right corner of your dashboard, everything is working!

**Location Summary:**
- **Page:** Student Dashboard (after login)
- **Position:** Top-right corner, next to profile picture
- **Icon:** Orange wrench (🔧)
- **Color:** Orange background with orange wrench
- **Size:** Square button, same size as profile button

---

## 💡 What Each Button Does

In Dev Tools screen:
| Button | Color | Purpose |
|--------|-------|---------|
| Seed Schools | Blue | Create test schools (run first) |
| Seed Test Data | Green | Create test data for student |
| Clear Test Data | Red | Delete test data |
| Open Latest Result | Orange | View last test result |
| 🎥 Test Media Upload | Green | **Test media upload system** ← USE THIS |

---

## ❌ Can't See the Wrench Icon?

**Solution:**
1. Make sure you ran `flutter run` after the latest changes
2. Try logging out and logging back in
3. Close and reopen the app completely
4. Check that you're on the Student Dashboard (not another page)
5. Look in the top-right area - it should be between the streak badge and profile icon

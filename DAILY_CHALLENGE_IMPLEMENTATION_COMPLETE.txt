🚀 HOW TO DEPLOY (3 STEPS):
========================================

STEP 1: Set Firebase Secret (ONE TIME)
---------------------------------------
cd cloudflare-worker
wrangler secret put FIREBASE_SERVICE_ACCOUNT --config wrangler-daily-challenge.jsonc
[Paste your Firebase service account JSON]

STEP 2: Deploy Worker
----------------------
wrangler deploy --config wrangler-daily-challenge.jsonc

STEP 3: Test It
---------------
curl -X POST https://daily-challenge-worker.YOUR_SUBDOMAIN.workers.dev

Then check Firebase Console → daily_challenges collection

========================================
🎓 HOW IT WORKS:
========================================

BEFORE (Current):
  Student opens app
    → Calls OpenTriviaDB API
    → Wait 5 seconds (rate limit)
    → Gets question
    → 1000 students = API overload ❌

AFTER (New System):
  2:00 AM IST (daily):
    → Worker fetches 3 questions
    → Stores in Firebase
  
  Student opens app:
    → Reads from Firebase (instant) ✅
    → No API call needed
    → No rate limits

========================================
💰 COST ANALYSIS:
========================================

OpenTriviaDB API:
  - Rate Limit: 5 seconds between requests
  - Cost: FREE
  - Problem: Too slow for many students ❌

Firebase Functions:
  - Cost: $25/month minimum
  - Problem: Requires paid subscription ❌

Cloudflare Workers:
  - Cost: FREE (100,000 requests/day)
  - Usage: 1 request per day
  - Perfect! ✅

Total Monthly Cost: $0 🎉

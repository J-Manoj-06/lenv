# Group Chat - Quick Start Guide

## 🚀 For Students

### Accessing Groups
1. Open the app and log in as a student
2. On the dashboard, look for the **orange Groups icon** (👥) in the top right
3. Tap the Groups icon

### Viewing and Joining Groups
- You'll see all subjects for your class (e.g., Grade 10 - Section B)
- Each subject shows:
  - Subject icon (🔢 📖 🔬 etc.)
  - Subject name
  - Teacher's name
- Tap any subject to open the group chat

### Sending Messages
1. Type your message in the text box at the bottom
2. Tap the orange send button (✉️)
3. Your message appears on the right side in an orange bubble

### Sending Images
1. Tap the image icon (🖼️) next to the text box
2. Select an image from your gallery
3. Image uploads automatically and appears in the chat

### Reading Messages
- **Your messages**: Orange bubbles on the right
- **Teacher's messages**: Dark bubbles on the left with teacher's name
- **Classmates' messages**: Dark bubbles on the left with their names
- Each message shows the sender's name and timestamp

## 👨‍🏫 For Teachers

### Accessing Groups
1. Open the app and log in as a teacher
2. On the dashboard, look for the **purple Groups icon** (👥) in the top right
3. Tap the Groups icon

### Viewing Your Groups
- You'll see all classes and subjects you teach
- Each group shows:
  - Subject icon
  - Subject name (in CAPS)
  - Class and section (e.g., Grade 10 - Section B)
- Groups are organized by class and section

### Opening a Group
- Tap any group card to open that subject's chat
- You'll see all messages from students in that class

### Sending Messages
1. Type your message in the text box at the bottom
2. Tap the orange send button
3. Your message appears on the right side
4. **All students** in that class section will see it instantly

### Sending Images
1. Tap the image icon next to the text box
2. Select an image from your gallery
3. Image uploads and all students see it

### Reading Student Messages
- All student messages appear on the left with dark background
- Each message shows the student's full name
- You can see messages from all students in the group
- Messages update in real-time

---

## 💡 Tips for Both

### Message Features
- ✅ **Real-time**: Messages appear instantly, no refresh needed
- ✅ **Auto-scroll**: Chat scrolls to latest message automatically
- ✅ **Timestamps**: See when each message was sent
- ✅ **Images**: Share photos with the group
- ✅ **Pull-to-refresh**: Swipe down on groups list to refresh

### Message Colors
- **Orange bubble (right)**: Your own messages
- **Dark bubble (left)**: Messages from others
- **Names shown**: See who sent each message

### Best Practices
1. **Be Respectful**: Remember it's a classroom environment
2. **Stay on Topic**: Keep discussions related to the subject
3. **Ask Questions**: Use the chat for doubts and clarifications
4. **Share Resources**: Images can be used to share notes, diagrams
5. **Check Regularly**: Important announcements may be posted

2. **Reminders**: "Don't forget your assignments"
3. **Answer Questions**: Help students with doubts
4. **Share Resources**: Post study materials, diagrams
5. **Class Updates**: "Test results will be out tomorrow"

## 🔍 Troubleshooting

### "No groups available"
**Student**: Check if you're assigned to a class
**Teacher**: Verify you're assigned to teach subjects

### "Messages not sending"
- Check internet connection
- Make sure you're logged in
- Try closing and reopening the chat

### "Groups button not visible"
- Make sure you're logged in
- Check you're on the correct dashboard (student/teacher)
- Try restarting the app

### "Can't see classmates' messages"
**Student**: You might be the first one in the chat!
**Teacher**: Students may not have sent messages yet
